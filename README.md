# DevTools do Henrique
Ferramentas online, rápidas e gratuitas.

## 🚀 Rodando localmente
```bash
npm install
npm run dev
```
Acesse: http://localhost:3000

## ☁️ Deploy na Vercel
1. Crie um repositório no GitHub.
2. Conecte à Vercel.
3. Deploy automático a cada push.
